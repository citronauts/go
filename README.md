# GridMash

A modern, responsive website built with HTML and CSS, hosted on Vercel.

## Features

- Clean, modern design with grid-based aesthetic
- Fully responsive layout
- Smooth animations and hover effects
- Fast loading with pure HTML/CSS

## Deployment

This site is configured for deployment on Vercel. Simply connect your repository to Vercel and it will automatically deploy.

### Quick Deploy

1. Fork or clone this repository
2. Connect to [Vercel](https://vercel.com)
3. Import the project
4. Deploy!

## Local Development

Simply open `index.html` in your browser to view the site locally.
